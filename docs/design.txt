More Complex (Events)
============

Create Event (name date) return id
   Create an event item
   Add attendees           - populate event_participant
   Add plates              - populate event_plate
   Add menu items          - populate event_menu_item


Edit Event (id, name, date)
==========
Change name, change date, add/remove attendees, add/remove plates and menu items.

Delete Event (id)
============
Delete enent, event participants, event plates and event menu items

More Complex (Consumption)
============

Add a plate for a given event and event participant
Remove a plate for a given event and event participant

Add a menu item for a given event and event participant
Remove a menu item for a given event and event participant
